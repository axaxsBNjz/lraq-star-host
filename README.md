# نظام إدارة مشتركي عراق ستار

## نظام احترافي لإدارة مشتركي خدمات الاستضافة

### المميزات:
- ✅ واجهة أمامية احترافية (Front-End Only)
- ✅ لوحة تحكم متقدمة للمسؤول
- ✅ بوابة خاصة بالمشتركين
- ✅ نظام إدارة المشتركين والدفعات
- ✅ تصميم متجاوب (Responsive Design)
- ✅ واجهة عربية كاملة

### الملفات:
- `index.html` - صفحة تسجيل الدخول
- `admin.html` - لوحة تحكم المسؤول
- `user.html` - بوابة المشترك
- `home.html` - الصفحة الرئيسية
- `style.css` - ملف الأنماط الشامل
- `script.js` - ملف JavaScript للتفاعلية
- `logo.png` - شعار الشركة

### كيفية الاستخدام:
1. افتح ملف `home.html` في المتصفح
2. اختر الخيار المناسب (تسجيل دخول، لوحة مسؤول، أو بوابة مشترك)
3. استخدم اسم المستخدم `admin` للدخول إلى لوحة المسؤول

### ملاحظات:
- هذا النظام هو واجهة أمامية فقط (Front-End)
- جميع البيانات المعروضة هي بيانات توضيحية
- يمكن ربط هذا النظام بخادم خلفي (Back-End) لاحقاً

### التطوير:
تم تطوير هذا النظام باستخدام:
- HTML5
- CSS3
- JavaScript (Vanilla)
- Bootstrap Icons
